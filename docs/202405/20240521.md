## Leetcode daily

[Subsets](https://leetcode.com/problems/subsets/?envType=daily-question&envId=2024-05-21)

```py
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res = []
        sub = []
        def dfs(idx):
            if idx == len(nums):
                res.append(sub[:])
                return
            sub.append(nums[idx])
            dfs(idx + 1)
            sub.pop()
            dfs(idx + 1)
        dfs(0)

        return res
```
