## Leetcode daily

[Maximum Odd Binary Number](https://leetcode.com/problems/maximum-odd-binary-number/?envType=daily-question&envId=2024-03-01)

```py
class Solution:
    def maximumOddBinaryNumber(self, s: str) -> str:
        n = len(s)
        count = s.count('1')
        return '1' * (count - 1) + '0' * (n - count) + '1'
```
